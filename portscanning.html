Penetration Test Commands


Port scanning
Utilizzare il tool nmap per individuare le porte TCP e UDP aperte
nmap -p- -sU -sT [target_IP]

Parametri:
  • -sS → TCP SYN scan (scansione stealth)
  • -sT → TCP connect scan (completa connessione)
  • -sU → UDP scan
  • -sA → scan degli ACK per firewall detection
  • -sV → rileva le versioni dei servizi
  • -O → rileva il sistema operativo
  • -Pn → disabilita il ping, assume che gli host siano attivi

    • In base alle porte aperte rilevate effettuare i seguenti test:
  • Porta 21 : FTP - anonymous login
  • Porta 22 : SSH - brute force attack
  • Porta 80/443 : HTTP/HTTPS - web spidering
  • Porta 80/443 : HTTP/HTTPS - sql injection
  • Porta 135 : RPC - service enumeration
  • Porta 445 : SMB - anonymous login
  • Porta 445 : SMB - brute force attack
  • Porta 3389 : RDP - brute force attack

    • Se in fase di vulnerability scan sono state rilevate vulnerabilità per cui è pubblicamente disponibile un exploit eseguire i test di sfruttamento con Metasploit
